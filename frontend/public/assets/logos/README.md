Place optional team logo SVG files in this folder to override monogram badges.

File naming:
- Use lowercase kebab-case slugs, such as `ferrari.svg`, `mercedes.svg`, `red-bull-racing.svg`.
- Badge lookup path is `assets/logos/{slug}.svg`.

Usage notes:
- If a logo file exists, the badge renders that SVG.
- If a logo file is missing, the badge falls back to a flat monogram style.
- Badge geometry stays fixed for all teams, same size and radius.

Trademark note:
Official Formula 1 team logos are trademarked. For a personal portfolio project, use simplified or monochrome marks that you have rights to use, then drop them here.
